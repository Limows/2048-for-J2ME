2048 for J2ME is a clone of the famous 2048 game for Java ME based cell phones.

It currently supports cell phones with standard keypad meeting these requirements:

  • 240x320 color screen
  • MIDP 2.0
  • CLDC 1.1
  • 2 MiB RAM
  • 100 KiB of storage
